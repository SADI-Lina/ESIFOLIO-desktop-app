Aide

Comment s'authentifier?

Pour s'authentifier, il suffit d'insérer votre adresse mail de l'ESI (suivant la syntaxe xx_xxxxxx@esi.dz) puis le code secret qui vous y sera envoyé.

Quelle syntaxe respecter lors de l'introduction des informations personnelles ?

Le nom et prénom doivent impérativement être insérés avec une première lettre majuscule. L'adresse mail doit correspondre à la syntaxe d'une adresse mail de l'ESI. Le matricule doit respecter le format 00/0000 et la date de naissance le format JJ/MM/AAAA. Pour les champs facultatifs, le numéro de téléphone doit tre constitué de 10 chiffres commençant par 0 et l'adresse doit être textuelle.

Comment sécuriser le répertoire contenant les portfolios générés ?

Pour sécuriser le répertoire contenant vos fichiers personnels, il suffit de choisir un mot de passe après s'être authentifié. Pour accéder au répertoire sécurisé, il faudra réintroduire ce mot de passe.

Comment modifier ses informations personnelles ?

Pour modifier ses informations personnelles, il suffit de cliquer sur l'icône située en haut à droite de la page d'accueil et de chaque page de l'application. Les informations personnelles actuelles y seront affichées, pour modifier un des champs il suffit de le réécrire et de cliquer sur

Comment créer un projet ?

Pour créer un projet, il suffit de sélectionner la rubrique projets se situant sur la barre latérale à gauche de l'application et de cliquer sur

Comment afficher les informations relatives à un projet ?

Pour afficher les informations relatives à un projet, après avoir sélectionné la rubrique projets , il suffit de cliquer sur le situé à droite du nom du projet concerné.

Comment modifier un projet ?

Pour modifier un projet préalablement créé, il suffit d'afficher les informations qui lui sont relatives et de modifier les champs désirés, hormis son type, le module concerné s'il s'agit d'un projet de l'école et le nom du club s'il s'agit d'un projet de club. Pour enregistrer les modifications, cliquer sur

Comment supprimer un projet ?

Pour supprimer un projet quelconque, il suffit de cliquer sur l'icône située à droite du nom du projet en question.

Comment afficher l'ensemble des compétences acquises ?

Pour afficher les compétences acquises à travers l'intégralité de vos projets réalisés, il suffit de se diriger vers la rubrique compétences pour les y retrouver toutes listées.

Comment télécharger le portfolio et où le retrouver ?

Pour télécharger le portfolio, il suffit de se diriger vers la rubrique et, après avoir choisi un des 2 modèles, cliquez sur pour télécharger votre portfolio en format HTML et/ou sur pour le télécharger en format PDF. Vous le retrouverez dans le répertoire sélectionné lors de la première connexion.

Quelles sont les statistiques pouvant être retrouvées sur l'application ?

Au niveau de la rubrique Statistiques , l'application permet de générer des statistiques concernant le nombre de projets réalisés et leur classification par type ainsi que le nombre de compétences acquises et leur classification par type.

Comment nous contacter en cas de problème ?

Afin de nous contacter, il suffit de cliquer sur située en bas à gauche de chaque page de l'application. Votre nom et adresse mail y seront immédiatement insérés et vous pourrez alors introduire un message expliquant le problème rencontré. Nous essayerons de vous répondre dans les plus brefs délais.
